## F. Pertanyaan Refleksi

1. Kapan sebaiknya menggunakan CSS Module dibanding Global CSS? <br>

- Digunakan ketika styling hanya untuk komponen tertentu dan ingin menghindari bentrok antar class. <br>

2. Apa kelemahan inline styling? <br>

- Tidak mendukung pseudo-class, media query, dan sulit dikelola pada project besar. Selain itu, kode menjadi kurang rapi karena styling bercampur dengan logika komponen. <br>

3. Mengapa SCSS cocok untuk project skala besar? <br>

- Mendukung variabel, nesting, dan mixin sehingga memudahkan pengelolaan dan konsistensi styling. Hal ini membuat kode lebih terstruktur dan mudah dirawat pada project besar. <br>

4. Apa keunggulan Tailwind dibanding CSS tradisional? <br>

- Tailwind mempercepat proses styling karena menggunakan utility class langsung di HTML. Lebih konsisten, minim konflik class, dan efisien untuk pengembangan UI modern. <br>
